

Para usar el programa se pueden ingresar dos paramentros de entrada por teclado 
para crear objetos aleatorios que están  limitados a un tamaño maximo de 1/3 el ancho de la caja.

Al iniciar el programa se debe  especificar  la cantidad de elementos a insertar y el otro el ancho de la caja.

Nota:  los objetos creados se imprimen en pantalla , para luego mostrar el ordenamiento logrado por nivel con el algoritmo implementado  programación Dinamica además se muestra el espacio restante por nivel.
